# App_Galeria_Fotos
- Funcionalidades Principais

1. O sistema exibe uma galeria de imagens obtidas em tempo real por meio da API do Pixabay:
2. Busca automática inicial com imagens de natureza;
3. Pesquisa de imagens por palavra-chave informada pelo usuário;
4. Exibição de imagens em layout responsivo;
5. Visualização de estatísticas das imagens (curtidas e visualizações);
6. Redirecionamento para a página original da imagem no Pixabay;
7. Animações suaves para melhor experiência do usuário;

- Características Técnicas

1. Integração direta com a API do Pixabay utilizando JavaScript;
2. Interface moderna desenvolvida com HTML5 e CSS3;
3. Design totalmente responsivo para desktop, tablet e dispositivos móveis;

- Requisitos

1. Chave de API válida do Pixabay;
2. Conexão com a internet para consumo da API;
3. Compativel com Windows, Linux e MacOS
